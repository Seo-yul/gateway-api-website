лӘ©лЎқ

Gateway APIмқҳ лӢӨмҡҙмҠӨнҠёлҰј кө¬нҳ„мІҙ л°Ҹ нҶөн•©

мқҙ л¬ём„ңлҠ” кІҢмқҙнҠёмӣЁмқҙ APIмқҳ лӢӨмҡҙмҠӨнҠёлҰј кө¬нҳ„ л°Ҹ нҶөн•©мқ„ м¶”м Ғн•ҳкі  мқҙл“Өм—җ лҢҖн•ң мғҒнғң л°Ҹ лҰ¬мҶҢмҠӨ м°ёмЎ°лҘј м ңкіөн•ңлӢӨ.

кІҢмқҙнҠёмӣЁмқҙ APIмқҳ кө¬нҳ„мһҗмҷҖ нҶөн•©мһҗл“ӨмқҖ мһҗмӢ л“Өмқҳ кө¬нҳ„мІҙм—җ лҢҖн•ң мғҒнғң м •ліҙ, м§Җмӣҗн•ҳлҠ” лІ„м „, к·ёлҰ¬кі  мӮ¬мҡ©мһҗк°Җ мӢңмһ‘н•ҳлҠ” лҚ° лҸ„мӣҖмқҙ лҗҳлҠ” л¬ём„ңлҘј, мқҙ л¬ём„ңм—җ м—…лҚ°мқҙнҠён•ҳлҠ” кІғмқҙ к¶ҢмһҘлҗңлӢӨ. мқҙ мғҒнғң м •ліҙлҠ” лӘҮ лӢЁлқҪмқ„ л„ҳм§Җ м•Ҡм•„м•ј н•ңлӢӨ.

нҳёнҷҳм„ұ(Conformance) мҲҳмӨҖ

кІҢмқҙнҠёмӣЁмқҙ API нҳёнҷҳм„ұм—җлҠ” м„ё к°Җм§Җ мҲҳмӨҖмқҙ мһҲлӢӨ.

нҳёнҷҳ(Conformant) кө¬нҳ„мІҙ

мқҙ кө¬нҳ„мІҙл“ӨмқҖ лӢӨмқҢ н•ӯлӘ©м—җ лҢҖн•ҙ нҶөкіјн•ң нҳёнҷҳм„ұ ліҙкі м„ңлҘј мөңмҶҢ н•ҳлӮҳ мқҙмғҒ м ңм¶ңн•ҳмҳҖлӢӨ.

  • н•ҳлӮҳ мқҙмғҒмқҳ Route нғҖмһ…кіј н”„лЎңн•„(Profile) мЎ°н•©м—җ лҢҖн•ң лӘЁл“  мҪ”м–ҙ(Core) нҳёнҷҳм„ұ н…ҢмҠӨнҠё
  • мЈјмһҘн•ҳлҠ” лӘЁл“  нҷ•мһҘ(Extended) кё°лҠҘ

к°ҖмһҘ мөңк·ј л‘җ(2)к°ңмқҳ кІҢмқҙнҠёмӣЁмқҙ API лҰҙлҰ¬мҠӨ мӨ‘ н•ҳлӮҳм—җ лҢҖн•ҙ мң„ мЎ°кұҙмқ„ л§ҢмЎұн•ҙм•ј н•ңлӢӨ.

мҰү, Mesh + HTTPRoute, Gateway + HTTPRoute, Gateway + TLSRoute, Gateway + Mesh + HTTPRoute л“ұмқ„ м§Җмӣҗн•ҳлҠ” кІғмқҙ нҳёнҷҳмқҙл©°, кө¬нҳ„мІҙк°Җ мЈјмһҘн•ҳлҠ” нҷ•мһҘ кё°лҠҘлҸ„ нҸ¬н•ЁлҗңлӢӨ. лӢЁ, кө¬нҳ„мІҙлҠ” л°ҳл“ңмӢң мөңмҶҢ н•ҳлӮҳмқҳ н”„лЎңн•„кіј н•ҙлӢ№ н”„лЎңн•„ лӮҙ н•ҳлӮҳмқҳ Route нғҖмһ…мқ„ м§Җмӣҗн•ҙм•ј н•ҳл©°, н•ҙлӢ№ н”„лЎңн•„кіј Route нғҖмһ…м—җ лҢҖн•ң лӘЁл“  мҪ”м–ҙ нҳёнҷҳм„ұ н…ҢмҠӨнҠёмҷҖ мЈјмһҘн•ҳлҠ” лӘЁл“  нҷ•мһҘ кё°лҠҘмқ„ нҶөкіјн•ҙм•ј н•ңлӢӨ.

л¶Җл¶„ нҳёнҷҳ(Partially Conformant) кө¬нҳ„мІҙ

мқҙ кө¬нҳ„мІҙл“ӨмқҖ мҷ„м „н•ң нҳёнҷҳмқ„ лӘ©н‘ңлЎң н•ҳкі  мһҲм§Җл§Ң нҳ„мһ¬ лӢ¬м„ұн•ҳм§Җ лӘ»н•ң мғҒнғңмқҙлӢӨ. к°ҖмһҘ мөңк·ј м„ё(3)к°ңмқҳ кІҢмқҙнҠёмӣЁмқҙ API лҰҙлҰ¬мҠӨ мӨ‘ н•ҳлӮҳм—җ лҢҖн•ҙ нҳёнҷҳ(мң„ кё°мӨҖ)мқҙ лҗҳкё° мң„н•ң мқјл¶Җ н…ҢмҠӨнҠёлҘј нҶөкіјн•ң нҳёнҷҳм„ұ ліҙкі м„ңлҘј мөңмҶҢ н•ҳлӮҳ мқҙмғҒ м ңм¶ңн•ҳмҳҖлӢӨ. "л¶Җл¶„ нҳёнҷҳ"мңјлЎң к°„мЈјлҗҳкё° мң„н•ң мҡ”кө¬ мӮ¬н•ӯмқҖ н–Ҙнӣ„ кІҢмқҙнҠёмӣЁмқҙ API лҰҙлҰ¬мҠӨм—җм„ң к°•нҷ”лҗ  мҲҳ мһҲлӢӨ.

л№„нҷңм„ұ(Stale) кө¬нҳ„мІҙ

мқҙ кө¬нҳ„мІҙл“ӨмқҖ нҳ„мһ¬ нҷңл°ңн•ҳкІҢ к°ңл°ңлҗҳкі  мһҲм§Җ м•Ҡмқ„ мҲҳ мһҲмңјл©°, лӢӨмқҢ нҺҳмқҙм§Җ кІҖнҶ  мӢң лӢӨлҘё м№ҙн…Ңкі лҰ¬лЎң мқҙлҸҷн•ҳлҠ” нҳёнҷҳм„ұ ліҙкі м„ңлҘј м ңм¶ңн•ҳм§Җ м•Ҡмңјл©ҙ мқҙ нҺҳмқҙм§Җм—җм„ң м ңкұ°лҗ  кІғмқҙлӢӨ.

нҺҳмқҙм§Җ кІҖнҶ лҠ” лӘЁл“  кІҢмқҙнҠёмӣЁмқҙ API лҰҙлҰ¬мҠӨ мқҙнӣ„ мөңмҶҢ 1к°ңмӣ” нӣ„м—җ мҲҳн–үлҗҳл©°, мІ« лІҲм§ё кІҖнҶ лҠ” кІҢмқҙнҠёмӣЁмқҙ API v1.3 лҰҙлҰ¬мҠӨ мқҙнӣ„ 2025л…„ 6мӣ” л§җм—җ мҲҳн–үлҗҳм—ҲлӢӨ. кІҢмқҙнҠёмӣЁмқҙ API v1.5 кІҖнҶ  н”„лЎңм„ёмҠӨ(2026л…„ мӨ‘л°ҳ мҳҲм •) мқҙнӣ„м—җлҠ” л№„нҷңм„ұ кө¬нҳ„мІҙк°Җ лҚ” мқҙмғҒ лӘ©лЎқм—җ н‘ңмӢңлҗҳм§Җ м•ҠлҠ”лӢӨ.

кө¬нҳ„ н”„лЎңн•„(Implementation profiles)

кө¬нҳ„мІҙлҠ” мқјл°ҳм ҒмңјлЎң н”„лЎңн•„ мқҙлқјкі  н•ҳлҠ” л‘җ к°Җм§Җ лІ”мЈјм—җ н•ҙлӢ№н•ңлӢӨ.

  • Gateway м»ЁнҠёлЎӨлҹ¬лҠ” Gateway лҰ¬мҶҢмҠӨлҘј мЎ°м •н•ҳл©°, мЈјлЎң нҒҙлҹ¬мҠӨн„° мҷёл¶Җм—җм„ң лӮҙл¶ҖлЎң л“Өм–ҙмҳӨлҠ” л¶Ғ/лӮЁ(north-south) нҠёлһҳн”Ҫмқ„ мІҳлҰ¬н•ҳкё° мң„н•ң кІғмқҙлӢӨ.
  • Mesh м»ЁнҠёлЎӨлҹ¬лҠ” HTTPRouteк°Җ м—°кІ°лҗң Service лҰ¬мҶҢмҠӨлҘј мЎ°м •н•ҳл©°, лҸҷмқј нҒҙлҹ¬мҠӨн„° лҳҗлҠ” нҒҙлҹ¬мҠӨн„° м§‘н•© лӮҙмқҳ лҸҷ/м„ң(east-west) нҠёлһҳн”Ҫмқ„ мІҳлҰ¬н•ҳкё° мң„н•ң кІғмқҙлӢӨ.

к°Ғ н”„лЎңн•„м—җлҠ” кө¬нҳ„мІҙк°Җ нҳёнҷҳ(мң„ кё°мӨҖ)мқҙ лҗҳкё° мң„н•ҙ кё°лҢҖлҗҳлҠ” лҸҷмһ‘мқ„ м •мқҳн•ҳлҠ” нҳёнҷҳм„ұ н…ҢмҠӨнҠё м„ёнҠёк°Җ м—°кІ°лҗҳм–ҙ мһҲлӢӨ.

кө¬нҳ„мІҙлҠ” л‘җ н”„лЎңн•„ лӘЁл‘җм—җ н•ҙлӢ№н•  мҲҳлҸ„ мһҲлӢӨ.

нҶөн•©(Integrations)

мқҙ нҺҳмқҙм§Җм—җлҠ” кІҢмқҙнҠёмӣЁмқҙ API лҰ¬мҶҢмҠӨлҘј нҷңмҡ©н•ҳм—¬ лӢӨлҘё кё°лҠҘ(DNS кҙҖлҰ¬ лҳҗлҠ” мқёмҰқм„ң мғқм„ұ л“ұ)мқ„ мҲҳн–үн•  мҲҳ мһҲлҠ” лӢӨлҘё мҶҢн”„нҠёмӣЁм–ҙ н”„лЎңм қнҠёмқё нҶөн•©(integrations) лҸ„ лӮҳм—ҙлҗҳм–ҙ мһҲлӢӨ.

м°ёкі :

мқҙ нҺҳмқҙм§Җм—җлҠ” кІҢмқҙнҠёмӣЁмқҙ APIк°Җ мһ‘лҸҷн•ҳлҠ” лҚ° н•„мҡ”н•ң кё°лҠҘмқ„ м ңкіөн•ҳлҠ” м„ңл“ң нҢҢнӢ° н”„лЎңм қнҠём—җ лҢҖн•ң л§ҒнҒ¬к°Җ нҸ¬н•Ёлҗҳм–ҙ мһҲлӢӨ. кІҢмқҙнҠёмӣЁмқҙ API н”„лЎңм қнҠё мһ‘м„ұмһҗлҠ” мқҙлҹ¬н•ң н”„лЎңм қнҠём—җ лҢҖн•ҙ мұ…мһ„м§Җм§Җ м•Ҡмңјл©°, н•ҙлӢ№ нҒҙлһҳмҠӨ лӮҙм—җм„ң м•ҢнҢҢлІімҲңмңјлЎң лӮҳм—ҙлҗҳм–ҙ мһҲлӢӨ.

м°ёкі :

кө¬нҳ„мІҙ к°„ нҷ•мһҘ м§Җмӣҗ кё°лҠҘ л№„көҗ

н”„лЎңм қнҠёмқҳ м§Җмӣҗ кё°лҠҘмқ„ л№ лҘҙкІҢ л№„көҗн•  мҲҳ мһҲлҠ” н‘ң нҷ•мқё. мқҙ н‘ңлҠ” мҪ”м–ҙ нҳёнҷҳм„ұ н…ҢмҠӨнҠёлҘј нҶөкіјн•ң кІҢмқҙнҠёмӣЁмқҙ м»ЁнҠёлЎӨлҹ¬ кө¬нҳ„мІҙлҘј к°ңмҡ”лЎң м ңмӢңн•ҳл©°, кө¬нҳ„н•ң нҷ•мһҘ нҳёнҷҳм„ұ кё°лҠҘм—җ мҙҲм җмқ„ л§һм¶ҳлӢӨ. мқҙ н‘ңлҠ” мөңмҶҢ 3к°ңмқҳ кө¬нҳ„мІҙк°Җ нҳёнҷҳм„ұ ліҙкі м„ң н•ҳмң„м—җ ліҙкі м„ңлҘј м—…лЎңл“ңн•ң нӣ„ мғқм„ұлҗҳм–ҙ мӮ¬мқҙнҠём—җ м—…лЎңл“ңлҗңлӢӨ.

AWS Load Balancer ControllerлҠ” мҝ лІ„л„ӨнӢ°мҠӨ нҒҙлҹ¬мҠӨн„°лҘј мң„н•ң AWS Elastic Load BalancerлҘј кҙҖлҰ¬н•ңлӢӨ. мқҙ м»ЁнҠёлЎӨлҹ¬лҠ” мҝ лІ„л„ӨнӢ°мҠӨ IngressлҘј мғқм„ұн•  л•Ң AWS Application Load Balancer(ALB)лҘј н”„лЎңл№„м ҖлӢқн•ҳкі , LoadBalancer нғҖмһ…мқҳ мҝ лІ„л„ӨнӢ°мҠӨ м„ңл№„мҠӨлҘј мғқм„ұн•  л•Ң AWS Network Load Balancer(NLB)лҘј н”„лЎңл№„м ҖлӢқн•ңлӢӨ.

  • FSMм—җм„ң кІҢмқҙнҠёмӣЁмқҙ API м§Җмӣҗмқ„ мӮ¬мҡ©н•ҳлҠ” л°©лІ•

Istioмқҳ мөңмҶҢ м„Өм№ҳл§ҢмңјлЎң нҒҙлҹ¬мҠӨн„° мқёк·ёл ҲмҠӨ нҠёлһҳн”Ҫ м ңм–ҙлҘј мң„н•ң мҝ лІ„л„ӨнӢ°мҠӨ кІҢмқҙнҠёмӣЁмқҙ API мҷ„м „н•ң м Ғн•© кө¬нҳ„мқ„ мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ. м„ңл№„мҠӨ л©”мӢң мӮ¬мҡ©мһҗлҘј мң„н•ҙ, IstioлҠ” л©”мӢң лӮҙм—җм„ң GAMMA мқҙлӢҲм…”нӢ°лёҢмқҳ кІҢмқҙнҠёмӣЁмқҙ API лҸҷ/м„ң нҠёлһҳн”Ҫ кҙҖлҰ¬ м§ҖмӣҗлҸ„ мҷ„м „нһҲ м§Җмӣҗн•ңлӢӨ.

лӘЁл“  мқёк·ёл ҲмҠӨ мһ‘м—…кіј м—¬лҹ¬ л©”мӢң лӮҙл¶Җ нҠёлһҳн”Ҫ кҙҖлҰ¬ мһ‘м—…мқ„ нҸ¬н•Ён•ң Istio л¬ём„ңмқҳ лҢҖл¶Җл¶„мқҖ мқҙлҜё кІҢмқҙнҠёмӣЁмқҙ API лҳҗлҠ” Istio кө¬м„ұ APIлҘј мӮ¬мҡ©н•ҳм—¬ нҠёлһҳн”Ҫмқ„ кө¬м„ұн•ҳлҠ” лі‘л ¬ м§Җм№Ёмқ„ нҸ¬н•Ён•ңлӢӨ. Istioмқҳ кІҢмқҙнҠёмӣЁмқҙ API кө¬нҳ„м—җ лҢҖн•ң мһҗм„ён•ң м •ліҙлҠ” кІҢмқҙнҠёмӣЁмқҙ API taskлҘј нҷ•мқён•ҳмһҗ.

Kong Kubernetes Ingress Controller

Kong кІҢмқҙнҠёмӣЁмқҙ мҳӨнҚјл Ҳмқҙн„°м—җ лҢҖн•ң лҸ„мӣҖкіј м§Җмӣҗмқ„ л°ӣмңјл Өл©ҙ мқҙмҠҲлҘј мғқм„ұн•ҳкұ°лӮҳ нҶ лЎ мқ„ л§Ңл“Өмһҗ. мҝ лІ„л„ӨнӢ°мҠӨ slackмқҳ #kong мұ„л„җм—җм„ңлҸ„ лҸ„мӣҖмқ„ мҡ”мІӯн•  мҲҳ мһҲлӢӨ.

NGINX Gateway Fabric

NGINX л¬ём„ң мӣ№мӮ¬мқҙнҠём—җм„ң мў…н•©м Ғмқё NGINX кІҢмқҙнҠёмӣЁмқҙ нҢЁлёҢлҰӯ мӮ¬мҡ©мһҗ л¬ём„ңлҘј м°ҫмқ„ мҲҳ мһҲлӢӨ.

м§ҖмӣҗлҗҳлҠ” кІҢмқҙнҠёмӣЁмқҙ API лҰ¬мҶҢмҠӨ л°Ҹ кё°лҠҘ лӘ©лЎқмқҖ кІҢмқҙнҠёмӣЁмқҙ API нҳёнҷҳм„ұ л¬ём„ңлҘј нҷ•мқён•ҳмһҗ.

NGINX кІҢмқҙнҠёмӣЁмқҙ нҢЁлёҢлҰӯм—җ лҢҖн•ң м ңм•Ҳмқҙ мһҲкұ°лӮҳ л¬ём ңлҘј кІҪн—ҳн–ҲлӢӨл©ҙ GitHubм—җм„ң мқҙмҠҲлҘј мғқм„ұн•ҳкұ°лӮҳ нҶ лЎ мқ„ л¶ҖнғҒн•ңлӢӨ. лҳҗн•ң NGINX м»Өл®ӨлӢҲнӢ° нҸ¬лҹјм—җм„ң лҸ„мӣҖлҸ„ мҡ”мІӯн•  мҲҳ мһҲлӢӨ.

STUNner

STUNnerлҠ” мҝ лІ„л„ӨнӢ°мҠӨмҡ© мҳӨн”ҲмҶҢмҠӨ нҒҙлқјмҡ°л“ң л„ӨмқҙнӢ°лёҢ WebRTC лҜёл””м–ҙ кІҢмқҙнҠёмӣЁмқҙмқҙлӢӨ. STUNnerлҠ” WebRTC лҜёл””м–ҙ мҠӨнҠёлҰјмқ„ мҝ лІ„л„ӨнӢ°мҠӨ нҒҙлҹ¬мҠӨн„°лЎң мӣҗнҷңн•ҳкІҢ мҲҳмӢ н•ҳкё° мң„н•ң лӘ©м ҒмңјлЎң м„Өкі„лҗҳм—Ҳмңјл©°, к°„мҶҢнҷ”лҗң NAT нҠёлһҳлІ„м„Өкіј лҸҷм Ғ лҜёл””м–ҙ лқјмҡ°нҢ…мқ„ м ңкіөн•ңлӢӨ. лҸҷмӢңм—җ STUNnerлҠ” лҢҖк·ңлӘЁ мӢӨмӢңк°„ нҶөмӢ  м„ңл№„мҠӨм—җ лҢҖн•ҙ ліҙм•Ҳм„ұкіј лӘЁлӢҲн„°л§Ғ кё°лҠҘмқ„ н–ҘмғҒмӢңнӮЁлӢӨ. STUNnerмқҳ лҚ°мқҙн„° н”Ңл ҲмқёмқҖ WebRTC нҒҙлқјмқҙм–ёнҠёлҘј мң„н•ң н‘ңмӨҖ к·ңкІ©мқҳ TURN м„ңл№„мҠӨлҘј м ңкіөн•ҳл©°, м»ЁнҠёлЎӨ н”Ңл ҲмқёмқҖ кІҢмқҙнҠёмӣЁмқҙ APIмқҳ мқјл¶ҖлҘј м§Җмӣҗн•ңлӢӨ.

нҳ„мһ¬ STUNnerлҠ” кІҢмқҙнҠёмӣЁмқҙ API лӘ…м„ёмқҳ v1alpha2 лІ„м „мқ„ м§Җмӣҗн•ңлӢӨ. WebRTC лҜёл””м–ҙ мҲҳмӢ мқ„ мң„н•ҙ STUNnerлҘј л°°нҸ¬н•ҳкі  мӮ¬мҡ©н•ҳлҠ” л°©лІ•мқҖ м„Өм№ҳ к°Җмқҙл“ңлҘј нҷ•мқён•ҳмһҗ. STUNnerмҷҖ кҙҖл Ёлҗң лӘЁл“  м§Ҳл¬ё, мқҳкІ¬ л°Ҹ лІ„к·ё лҰ¬нҸ¬нҠёлҠ” STUNner н”„лЎңм қнҠёлЎң ліҙлӮҙмЈјмӢңкё° л°”лһҖлӢӨ.

Traefik Proxy

Traefik ProxyлҠ” мҳӨн”Ҳ мҶҢмҠӨ нҒҙлқјмҡ°л“ң л„ӨмқҙнӢ°лёҢ м• н”ҢлҰ¬мјҖмқҙм…ҳ н”„лЎқмӢңмқҙлӢӨ.

Traefik н”„лЎқмӢңлҠ” нҳ„мһ¬ кІҢмқҙнҠёмӣЁмқҙ API лӘ…м„ёмқҳ v1.4.0 лІ„м „мқ„ м§Җмӣҗн•ңлӢӨ. л°°нҸ¬ л°Ҹ мӮ¬мҡ© л°©лІ•м—җ лҢҖн•ң мһҗм„ён•ң м •ліҙлҠ” мҝ лІ„л„ӨнӢ°мҠӨ кІҢмқҙнҠёмӣЁмқҙ н”„лЎңл°”мқҙлҚ” л¬ём„ңлҘј нҷ•мқён•ҳмһҗ. Traefik н”„лЎқмӢңмқҳ кө¬нҳ„мқҖ GRPCRouteмҷҖ к°ҷмқҖ лӘЁл“  HTTP мҪ”м–ҙ л°Ҹ мқјл¶Җ нҷ•мһҘ нҳёнҷҳм„ұ н…ҢмҠӨнҠёлҘј нҶөкіјн•ҳл©°, мӢӨн—ҳм Ғ мұ„л„җмқҳ TCPRoute л°Ҹ TLSRoute кё°лҠҘлҸ„ м§Җмӣҗн•ңлӢӨ.

Traefik н”„лЎқмӢңм—җ лҢҖн•ң лҸ„мӣҖкіј м§Җмӣҗмқ„ л°ӣмңјл Өл©ҙ, мқҙмҠҲлҘј мғқм„ұн•ҳкұ°лӮҳ Traefik Labs м»Өл®ӨлӢҲнӢ° нҸ¬лҹјм—җм„ң лҸ„мӣҖмқ„ мҡ”мІӯн•ҳмһҗ.

Tyk

Tyk кІҢмқҙнҠёмӣЁмқҙлҠ” нҒҙлқјмҡ°л“ң л„ӨмқҙнӢ°лёҢ мҳӨн”ҲмҶҢмҠӨ API кІҢмқҙнҠёмӣЁмқҙмқҙлӢӨ.

Tyk.io нҢҖмқҖ кІҢмқҙнҠёмӣЁмқҙ API кө¬нҳ„мқ„ лӘ©н‘ңлЎң мһ‘м—… мӨ‘мқҙл©°, мқҙ н”„лЎңм қнҠёмқҳ м§„н–ү мғҒнҷ©мқҖ м—¬кё°м—җм„ң нҷ•мқён•  мҲҳ мһҲлӢӨ.

WSO2 APK

WSO2 APKлҠ” мҝ лІ„л„ӨнӢ°мҠӨ нҷҳкІҪмқ„ мң„н•ҙ нҠ№лі„нһҲ м„Өкі„лҗң API кҙҖлҰ¬ мҶ”лЈЁм…ҳмңјлЎң, API кҙҖлҰ¬лҘј мң„н•ң нҶөн•©м„ұ, мң м—°м„ұ, нҷ•мһҘм„ұмқ„ мЎ°м§Ғм—җ м ңкіөн•ңлӢӨ.

WSO2 APKлҠ” кІҢмқҙнҠёмӣЁмқҙ APIлҘј кө¬нҳ„н•ҳл©°, кІҢмқҙнҠёмӣЁмқҙ л°Ҹ HTTPRoute кё°лҠҘмқ„ нҸ¬н•Ён•ңлӢӨ. лҳҗн•ң, мӮ¬мҡ©мһҗ м •мқҳ лҰ¬мҶҢмҠӨ(CR)лҘј нҶөн•ҙ л ҲмқҙнҠё лҰ¬л°ӢнҢ…, мқёмҰқ/мқёк°Җ, л¶„м„қ/кҙҖм°° к°ҖлҠҘм„ұмқ„ м§Җмӣҗн•ңлӢӨ.

кІҢмқҙнҠёмӣЁмқҙ APIмқҳ м§Җмӣҗ лІ„м „кіј кё°лҠҘм—җ лҢҖн•ң мөңмӢ  м •ліҙлҠ” APK кІҢмқҙнҠёмӣЁмқҙ л¬ём„ңлҘј м°ёкі н•ҳмһҗ. м§Ҳл¬ёмқҙ мһҲкұ°лӮҳ кё°м—¬н•ҳкі  мӢ¶лӢӨл©ҙ мһҗмң лЎӯкІҢ мқҙмҠҲ лҳҗлҠ” н’Җ лҰ¬нҖҳмҠӨнҠёлҘј мғқм„ұн•  мҲҳ мһҲлӢӨ. лҳҗн•ң Discord мұ„л„җм—җм„ң мҡ°лҰ¬мҷҖ мҶҢнҶөн•ҳкі  нҶ лЎ м—җ м°ём—¬н•  мҲҳ мһҲлӢӨ.

нҶөн•©

мқҙ м„№м…ҳм—җм„ңлҠ” нҠ№м • нҶөн•©мқ„ мң„н•ң лё”лЎңк·ё нҸ¬мҠӨнҠё, л¬ём„ң л°Ҹ кё°нғҖ кІҢмқҙнҠёмӣЁмқҙ API м°ёмЎ°м—җ лҢҖн•ң кө¬мІҙм Ғмқё л§ҒнҒ¬лҘј м°ҫмқ„ мҲҳ мһҲлӢӨ.

Flagger

FlaggerлҠ” мҝ лІ„л„ӨнӢ°мҠӨм—җм„ң мӢӨн–үлҗҳлҠ” м• н”ҢлҰ¬мјҖмқҙм…ҳмқҳ лҰҙлҰ¬мҠӨ н”„лЎңм„ёмҠӨлҘј мһҗлҸҷнҷ”н•ҳлҠ” м җм§„м Ғ л°°нҸ¬ лҸ„кө¬мқҙлӢӨ.

FlaggerлҠ” кІҢмқҙнҠёмӣЁмқҙ APIлҘј мӮ¬мҡ©н•ҳм—¬ м№ҙлӮҳлҰ¬ л°°нҸ¬мҷҖ A/B н…ҢмҠӨнҠёлҘј мһҗлҸҷнҷ”н•ҳлҠ” лҚ° мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ. кІҢмқҙнҠёмӣЁмқҙ APIмқҳ v1alpha2мҷҖ v1beta1 лӘ…м„ёлҘј лӘЁл‘җ м§Җмӣҗн•ңлӢӨ. кІҢмқҙнҠёмӣЁмқҙ APIмқҳ лӘЁл“  кө¬нҳ„кіј н•Ёк»ҳ FlaggerлҘј мӮ¬мҡ©н•ҳл Өл©ҙ мқҙ нҠңнҶ лҰ¬м–јмқ„ м°ёмЎ°н•ңлӢӨ.

cert-manager

cert-managerлҠ” нҒҙлқјмҡ°л“ң л„ӨмқҙнӢ°лёҢ нҷҳкІҪм—җм„ң мқёмҰқм„ң кҙҖлҰ¬лҘј мһҗлҸҷнҷ”н•ҳкё° мң„н•ң лҸ„кө¬мқҙлӢӨ.

cert-managerлҠ” кІҢмқҙнҠёмӣЁмқҙ лҰ¬мҶҢмҠӨлҘј мң„н•ң TLS мқёмҰқм„ңлҘј мғқм„ұн•  мҲҳ мһҲлӢӨ. мқҙлҠ” кІҢмқҙнҠёмӣЁмқҙм—җ м–ҙл…ён…Ңмқҙм…ҳмқ„ м¶”к°Җн•ҳм—¬ кө¬м„ұлҗңлӢӨ. нҳ„мһ¬ кІҢмқҙнҠёмӣЁмқҙ APIмқҳ v1 лӘ…м„ёлҘј м§Җмӣҗн•ңлӢӨ. мӮ¬мҡ©н•ҙліҙл Өл©ҙ cert-manager л¬ём„ңлҘј м°ёмЎ°н•ңлӢӨ.

Argo rollouts

Argo RolloutsлҠ” мҝ лІ„л„ӨнӢ°мҠӨлҘј мң„н•ң м җм§„м Ғ л°°нҸ¬ м»ЁнҠёлЎӨлҹ¬мқҙлӢӨ. лё”лЈЁ/к·ёлҰ° л°Ҹ м№ҙлӮҳлҰ¬мҷҖ к°ҷмқҖ м—¬лҹ¬ кі кёү л°°нҸ¬ л°©лІ•мқ„ м§Җмӣҗн•ңлӢӨ. Argo RolloutsлҠ” н”Ңлҹ¬к·ёмқёмқ„ нҶөн•ҙ кІҢмқҙнҠёмӣЁмқҙ APIлҘј м§Җмӣҗн•ңлӢӨ.

Knative

KnativeлҠ” мҝ лІ„л„ӨнӢ°мҠӨ мң„м—җ кө¬м¶•лҗң м„ңлІ„лҰ¬мҠӨ н”Ңлһ«нҸјмқҙлӢӨ. Knative ServingмқҖ URLмқҳ мһҗлҸҷ кҙҖлҰ¬, лҰ¬л№„м „ к°„ нҠёлһҳн”Ҫ л¶„н• , мҡ”мІӯ кё°л°ҳ мһҗлҸҷ мҠӨмјҖмқјл§Ғ(м ңлЎң мҠӨмјҖмқј нҸ¬н•Ё), мһҗлҸҷ TLS н”„лЎңл№„м ҖлӢқкіј н•Ёк»ҳ мғҒнғң л№„м ҖмһҘ м»Ён…Ңмқҙл„ҲлҘј мӢӨн–үн•ҳкё° мң„н•ң к°„лӢЁн•ң APIлҘј м ңкіөн•ңлӢӨ. Knative ServingмқҖ н”Ңлҹ¬к·ёмқё м•„нӮӨн…ҚмІҳлҘј нҶөн•ҙ лӢӨмӨ‘ HTTP лқјмҡ°н„°лҘј м§Җмӣҗн•ҳл©°, мқҙлҠ” лӘЁл“  Knative кё°лҠҘмқҙ м§Җмӣҗлҗҳм§Җ м•Ҡм•„ нҳ„мһ¬ м•ҢнҢҢ лӢЁкі„м—җ мһҲлҠ” кІҢмқҙнҠёмӣЁмқҙ API н”Ңлҹ¬к·ёмқёмқ„ нҸ¬н•Ён•ңлӢӨ.

Kuadrant

KuadrantлҠ” лӢӨлҘё кІҢмқҙнҠёмӣЁмқҙ API м ңкіөмһҗмҷҖ нҶөн•©лҗҳкі  м •мұ… м—°кІ°мқ„ нҶөн•ҙ м •мұ…мқ„ м ңкіөн•ҳлҠ” мҳӨн”Ҳ мҶҢмҠӨ л©ҖнӢ° нҒҙлҹ¬мҠӨн„° кІҢмқҙнҠёмӣЁмқҙ API м»ЁнҠёлЎӨлҹ¬мқҙлӢӨ.

KuadrantлҠ” кІҢмқҙнҠёмӣЁмқҙлҘј мӨ‘м•ҷм—җм„ң м •мқҳн•ҳкі  лӘЁл“  кІҢмқҙнҠёмӣЁмқҙм—җ м Ғмҡ©лҗҳлҠ” DNS, TLS, мқёмҰқ л°Ҹ л ҲмқҙнҠё лҰ¬л°ӢнҢ…кіј к°ҷмқҖ м •мұ…мқ„ м—°кІ°н•ҳкё° мң„н•ң кІҢмқҙнҠёмӣЁмқҙ APIлҘј м§Җмӣҗн•ңлӢӨ.

KuadrantлҠ” IstioмҷҖ Envoy GatewayлҘј кё°ліё кІҢмқҙнҠёмӣЁмқҙ API м ңкіөмһҗлЎң м§Җмӣҗн•ҳл©°, н–Ҙнӣ„ лӢӨлҘё кІҢмқҙнҠёмӣЁмқҙ м ңкіөмһҗмҷҖлҸ„ мһ‘лҸҷн•  кі„нҡҚмқҙлӢӨ.

Kuadrantмқҳ кө¬нҳ„м—җ лҢҖн•ң лҸ„мӣҖкіј м§Җмӣҗмқ„ л°ӣмңјл Өл©ҙ, мһҗмң лЎӯкІҢ мқҙмҠҲлҘј мғқм„ұн•ҳкұ°лӮҳ мҝ лІ„л„ӨнӢ°мҠӨ slackмқҳ #kuadrant мұ„л„җм—җм„ң лҸ„мӣҖмқ„ мҡ”мІӯн•ҳмһҗ.

OpenKruise Rollouts

OpenKruise RolloutsлҠ” мҝ лІ„л„ӨнӢ°мҠӨлҘј мң„н•ң н”Ңлҹ¬к·ё м•Ө н”Ңл Ҳмқҙ м җм§„м Ғ л°°нҸ¬ м»ЁнҠёлЎӨлҹ¬мқҙлӢӨ. лё”лЈЁ/к·ёлҰ° л°Ҹ м№ҙлӮҳлҰ¬мҷҖ к°ҷмқҖ м—¬лҹ¬ кі кёү л°°нҸ¬ л°©лІ•мқ„ м§Җмӣҗн•ңлӢӨ. OpenKruise RolloutsлҠ” кІҢмқҙнҠёмӣЁмқҙ APIм—җ лҢҖн•ң лӮҙмһҘ м§Җмӣҗмқ„ м ңкіөн•ңлӢӨ.

мғҲ н•ӯлӘ© м¶”к°Җ

кө¬нҳ„мІҙлҠ” мһҗмң лЎӯкІҢ PRмқ„ л§Ңл“Өм–ҙ мқҙ нҺҳмқҙм§Җм—җ н•ӯлӘ©мқ„ м¶”к°Җн•  мҲҳ мһҲлӢӨ. к·ёлҹ¬лӮҳ л¶Җл¶„ нҳёнҷҳ лҳҗлҠ” нҳёнҷҳ мҡ”кө¬ мӮ¬н•ӯмқ„ м¶©мЎұн•ҳл Өл©ҙ кө¬нҳ„мІҙмқҳ нҳёнҷҳм„ұ ліҙкі м„ң м ңм¶ң PRмқҙ лі‘н•©лҗҳм–ҙ мһҲм–ҙм•ј н•ңлӢӨ.

мқҙ нҺҳмқҙм§Җм—җ мғҲлЎң м¶”к°ҖлҗҳлҠ” н•ӯлӘ©м—җ лҢҖн•ң кІҖнҶ  н”„лЎңм„ёмҠӨмқҳ мқјл¶ҖлЎң, л©”мқён…Ңмқҙл„Ҳк°Җ нҳёнҷҳм„ұ мҲҳмӨҖмқ„ нҷ•мқён•ҳкі  мғҒнғңлҘј кІҖмҰқн•ңлӢӨ.

нҺҳмқҙм§Җ кІҖнҶ  м •мұ…

мқҙ нҺҳмқҙм§ҖлҠ” нҷңл°ңнһҲ к°ңл°ңлҗҳкі  мһҲмңјл©° нҳёнҷҳлҗҳлҠ” кІҢмқҙнҠёмӣЁмқҙ API кө¬нҳ„мІҙлҘј ліҙм—¬мЈјкё° мң„н•ң кІғмқҙл©°, м •кё°м Ғмқё кІҖнҶ  лҢҖмғҒмқҙлӢӨ.

мқҙлҹ¬н•ң кІҖнҶ лҠ” лӘЁл“  кІҢмқҙнҠёмӣЁмқҙ API лҰҙлҰ¬мҠӨ мқҙнӣ„ мөңмҶҢ 1к°ңмӣ” нӣ„м—җ мҲҳн–үлҗңлӢӨ (кІҢмқҙнҠёмӣЁмқҙ API v1.3 лҰҙлҰ¬мҠӨл¶Җн„° мӢңмһ‘).

кІҖнҶ мқҳ мқјл¶ҖлЎң, л©”мқён…Ңмқҙл„ҲлҠ” лӢӨмқҢмқ„ нҷ•мқён•ңлӢӨ.

  • мқҙ л¬ём„ңм—җм„ң мң„м—җ м •мқҳлҗң лҢҖлЎң нҳёнҷҳ(Conformant) мқё кө¬нҳ„мІҙ
  • мқҙ л¬ём„ңм—җм„ң мң„м—җ м •мқҳлҗң лҢҖлЎң л¶Җл¶„ нҳёнҷҳ(Partially Conformant) мқё кө¬нҳ„мІҙ

кІҖнҶ лҘј мҲҳн–үн•ҳлҠ” л©”мқён…Ңмқҙл„Ҳк°Җ л¶Җл¶„ нҳёнҷҳ лҳҗлҠ” нҳёнҷҳ кё°мӨҖмқ„ лҚ” мқҙмғҒ м¶©мЎұн•ҳм§Җ м•ҠлҠ” кө¬нҳ„мІҙлҘј л°ңкІ¬н•ҳкұ°лӮҳ "л№„нҷңм„ұ(Stale)" мғҒнғңмқё кө¬нҳ„мІҙлҘј л°ңкІ¬н•ҳл©ҙ, н•ҙлӢ№ л©”мқён…Ңмқҙл„ҲлҠ” лӢӨмқҢмқ„ мҲҳн–үн•ңлӢӨ.

  • лӢӨлҘё л©”мқён…Ңмқҙл„Ҳм—җкІҢ м•ҢлҰ¬кі  л№„нҷңм„ұ л°Ҹ м ңкұ° мҳҲм • кө¬нҳ„мІҙ лӘ©лЎқм—җ лҢҖн•ң лҸҷмқҳлҘј л°ӣлҠ”лӢӨ.
  • мқҙ нҺҳмқҙм§Җмқҳ ліҖкІҪ мӮ¬н•ӯмқҙ нҸ¬н•Ёлҗң л“ңлһҳн”„нҠё PRмқ„ м—°лӢӨ.
  • #sig-network-gateway-api мұ„л„җм—җ кІҢмӢңн•ҳм—¬ мөңмҶҢн•ң л¶Җл¶„ нҳёнҷҳмқҙ м•„лӢҢ кө¬нҳ„мІҙмқҳ л©”мқён…Ңмқҙл„Ҳк°Җ кІҢмқҙнҠёмӣЁмқҙ API л©”мқён…Ңмқҙл„Ҳм—җкІҢ м—°лқҪн•ҳм—¬ кө¬нҳ„мІҙмқҳ мғҒнғңлҘј л…јмқҳн•ҙм•ј н•Ёмқ„ м•ҢлҰ°лӢӨ. мқҙ кё°к°„мқ„ "лӢөліҖ к¶ҢлҰ¬(right-of-reply)" кё°к°„мқҙлқј н•ҳл©° мөңмҶҢ 2мЈјмқҙкі , м§Җм—° н•©мқҳ(lazy consensus) кё°к°„мңјлЎң кё°лҠҘн•ңлӢӨ.
  • лӢөліҖ к¶ҢлҰ¬ кё°к°„ лӮҙм—җ мқ‘лӢөн•ҳм§Җ м•ҠлҠ” кө¬нҳ„мІҙлҠ” "л№„нҷңм„ұ"мңјлЎң мқҙлҸҷлҗҳкұ°лӮҳ, мқҙлҜё "л№„нҷңм„ұ"мқё кІҪмҡ° мқҙ нҺҳмқҙм§Җм—җм„ң м ңкұ°лҗҳм–ҙ мғҒнғңк°Җ н•ҳн–Ҙ мЎ°м •лҗңлӢӨ.

v1.4 нҺҳмқҙм§Җ кІҖнҶ л¶Җн„° мӢңмһ‘н•ҳлҠ” нҺҳмқҙм§Җ кІҖнҶ  мқјм •:

  • кІҢмқҙнҠёмӣЁмқҙ API v1.4 лҰҙлҰ¬мҠӨ нҺҳмқҙм§Җ кІҖнҶ (мӢӨм ң лҰҙлҰ¬мҠӨ мқҙнӣ„ мөңмҶҢ 1к°ңмӣ” нӣ„): л©”мқён…Ңмқҙл„Ҳк°Җ мң„ к·ңм№ҷм—җ л”°лқј нҳёнҷҳм„ұ ліҙкі м„ңлҘј м ңм¶ңн•ҳм§Җ м•ҠмқҖ кө¬нҳ„мІҙлҘј "л№„нҷңм„ұ"мңјлЎң мқҙлҸҷмӢңнӮЁлӢӨ. лҳҗн•ң л№„нҷңм„ұмңјлЎң мқҙлҸҷлҗҳлҠ” кө¬нҳ„мІҙм—җкІҢ мқҙ к·ңм№ҷ ліҖкІҪм—җ лҢҖн•ҙ м•ҢлҰ°лӢӨ. нҳ„мһ¬ мқҙ лӢЁкі„мқҙлӢӨ
  • кІҢмқҙнҠёмӣЁмқҙ API v1.5 лҰҙлҰ¬мҠӨ нҺҳмқҙм§Җ кІҖнҶ (мӢӨм ң лҰҙлҰ¬мҠӨ мқҙнӣ„ мөңмҶҢ 1к°ңмӣ” нӣ„): л©”мқён…Ңмқҙл„Ҳк°Җ нҺҳмқҙм§Җ кІҖнҶ  н”„лЎңм„ёмҠӨлҘј лӢӨмӢң мҲҳн–үн•ҳм—¬ м—¬м „нһҲ л№„нҷңм„ұмқё кө¬нҳ„мІҙлҘј м ңкұ°н•ңлӢӨ(лӢөліҖ к¶ҢлҰ¬ кё°к°„ мқҙнӣ„).
  • кІҢмқҙнҠёмӣЁмқҙ API v1.6 лҰҙлҰ¬мҠӨ нҺҳмқҙм§Җ кІҖнҶ (мӢӨм ң лҰҙлҰ¬мҠӨ мқҙнӣ„ мөңмҶҢ 1к°ңмӣ” нӣ„): л№„нҷңм„ұ м№ҙн…Ңкі лҰ¬лҘј м ңкұ°н•ҳкі , кө¬нҳ„мІҙ л©”мқён…Ңмқҙл„ҲлҠ” к°Ғ кІҖнҶ  мӢң лҳҗлҠ” лӢөліҖ к¶ҢлҰ¬ кё°к°„ лӮҙм—җ мөңмҶҢн•ң л¶Җл¶„ нҳёнҷҳмқҙм–ҙм•ј н•ҳл©°, к·ёл Үм§Җ м•Ҡмңјл©ҙ кө¬нҳ„мІҙ нҺҳмқҙм§Җм—җм„ң м ңкұ°лҗңлӢӨ.

мқҙлҠ” кІҢмқҙнҠёмӣЁмқҙ API v1.6 лҰҙлҰ¬мҠӨ мқҙнӣ„м—җлҠ” мөңмҶҢн•ң л¶Җл¶„ нҳёнҷҳ(Partially Conformant) нҳёнҷҳм„ұ ліҙкі м„ңлҘј м ңм¶ңн•ҳм§Җ м•Ҡмңјл©ҙ мқҙ нҺҳмқҙм§Җм—җ кө¬нҳ„мІҙлҘј м¶”к°Җн•  мҲҳ м—ҶмқҢмқ„ мқҳлҜён•ңлӢӨ.
